# Strzykawa Site — AI Memory

## Project Summary
Coffee shop + roastery e-commerce. React 18 + Vite + Tailwind CSS + Shopify Storefront API + Zustand. Deployed on Netlify from `main` branch.

## Key Architecture
- **Entry:** `src/main.jsx` → `src/App.jsx` (React Router v6, 18 routes)
- **State:** Zustand stores in `src/store/` (cartStore, authStore, checkoutStore)
- **Shopify:** `src/services/shopify/` (client, products, cart, customer, mapper)
- **Pages:** `src/pages/` — 19 pages total
- **Components:** Atomic Design in `src/components/`
- **Constants:** `src/constants/` (navigation, colors, shipping, timings, contact, preview)

## SEO Implementation (added 2026-03-03)
- **Library:** `react-helmet-async` — installed, `<HelmetProvider>` wraps app in `main.jsx`
- **Component:** `src/components/SEO.jsx` — reusable, accepts: `title`, `description`, `canonical`, `ogImage`, `ogType`, `noindex`, `productSchema`, `fullTitle`
- **Pages with SEO:** Home, Coffees, CoffeeDetail (+ JSON-LD Product schema), About, Contact, B2B, TermsAndConditions, PrivacyPolicy, CookiePolicy
- **Pages with noindex:** Profile, Orders, CheckoutPage, NotFound
- **Sitemap:** `public/sitemap.xml` — already existed with all products; `public/robots.txt` already references it
- **JSON-LD on product pages:** `@type: Product` with name, description, image, brand, offers (price, availability, url)

## Design Rules (CRITICAL)
- Buttons/badges: `rounded-full` ONLY
- Count badges: `bg-success` (NOT `bg-accent`)
- Cards/Modals: NO rounding (sharp corners)
- Never add features beyond requirements

## Important Files
- `CLAUDE.md` — comprehensive project guide
- `DESIGN_SYSTEM.md` — design rules
- `tailwind.config.js` — custom color system (primary, accent, muted, success, danger, cta)
- `public/sitemap.xml` — already auto-generated by `scripts/generate-sitemap.mjs` on build
- `src/constants/contact.js` — centralized business contact data
